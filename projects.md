### 会议记录 AI Agent

**项目描述：**
一个自研的会议记录 AI Agent，能够自动识别会议内容，提取关键信息，并生成结构化的会议纪要。该项目使用了最新的大语言模型技术，能够理解会议中的对话，识别重要决策和行动项，并以清晰的格式呈现。

**技术栈：**
- Python
- OpenAI API
- 语音识别技术
- 自然语言处理

**主要功能：**
- 自动记录会议内容
- 提取关键信息和决策
- 生成结构化会议纪要
- 支持多种语言
- 可集成到常见的会议平台

**项目状态：** 开发中
